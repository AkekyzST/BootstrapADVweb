APPLICATION OVERVIEW

Classic login page

User can login to application for sell itemsa and manange their products.

Location-based menu

Pins on map shows trading pin that collect from Twitter and from application around user by using Google Maps API.

Add product page

User can add product by complete this form. User can add 5 maximum images per product.

Registeration page

Simply registeration form.

Support IOS/Android

We implemented this application as web application by use Ionic3 framework

Linked with Twitter

This is an one way advertising your sells. When you create trading post, system will post your sells to Twitter too.

Anything can sell

You can sell everything you don't need. You just post here, no need to go to market anymore.
